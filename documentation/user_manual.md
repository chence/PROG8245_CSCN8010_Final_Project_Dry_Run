# User Manual

## Run the baseline model
1. Install dependencies from `requirements.txt`
2. Run the training command
3. Run the evaluation command
4. Launch `app.py`

## Ask example questions
- I have a sore throat and cough
- What should I do for a mild fever
- Can I take medicine for congestion
- Should I see a doctor if breathing feels harder

## Known Limitations
- The current dataset is small and curated for a dry run
- Translation quality depends on the translation library or external service used
- The response layer uses templates rather than fully generated dialogue
